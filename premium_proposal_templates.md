## 共通ルール（全パターン適用）

| ルール | 内容 |
| --- | --- |
| 文字数 | **200〜350文字**（旧: 400〜550文字から圧縮） |
| 冒頭 | 「はじめまして」禁止。案件名への言及 or 課題への共感から始める |
| 段落上限 | 1段落は最大3行まで |
| 段落間 | 必ず1行空ける |
| 末尾固定 | 資格情報（Anthropic証明書URL）を必ず記載 |
| 禁止表現 | 「実績があります」「実績あり」→ Before/After or 「実稼働させています」で代替 |
| Before/After | 案件内容に即した具体数字を入れる（汎用記述禁止） |
| CTA | 「ご返信お待ちしております」禁止 → 「30分のヒアリングから始めませんか？」「どのような形でヒアリングが可能でしょうか？」等の具体的な次アクション |
| Claude Code言及 | AI・自動化系案件では必ず「Claude Code」を本文中に入れる |
